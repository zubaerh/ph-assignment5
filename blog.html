Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Gets/sets HTML markup| innerText: Gets/sets visible text| textContent: Gets/sets all text, including hidden.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you attach a single event listener to a parent element, instead of adding listeners to each of its child elements. When an event occurs on a child, it "bubbles up" to the parent, and the parent's listener handles it. This is very efficient.

Question-4: What is event bubbling in the DOM?

Imagine dropping a pebble in a pond; the ripples spread outward. Event bubbling is like those ripples, but going up the HTML element "pond"

Question-5: How do you create, add, and remove elements using JavaScript?

// Create an element:
const newElement = document.createElement('div');
// Add content:
newElement.textContent = 'This is a new div!';
// Add attributes:
newElement.setAttribute('id', 'myDiv');
newElement.classList.add('my-class');
// Add to the DOM:
const parentElement = document.getElementById('parent'); // Example parent
parentElement.appendChild(newElement); // Add as last child
// Add before another element:
const referenceElement = document.getElementById('reference');
parentElement.insertBefore(newElement, referenceElement);
// Remove an element: newElement.remove(); // Modern way
// or
parentElement.removeChild(newElement); // Older way, requires parent reference.
